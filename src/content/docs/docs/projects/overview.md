The Projects section is your central hub for tracking all active and completed work. BuildWorkPro offers multiple views so you can work the way you prefer.

## Project views

- **Board view** -- A Kanban-style board where projects are organized by status columns. Drag and drop projects between columns to update their status.
- **List view** -- A sortable table showing all projects with key details like client, status, contract value, and dates. Great for scanning a large number of projects.
- **Grid view** -- A card-based layout that shows project summaries at a glance, including progress indicators and upcoming milestones.

Switch between views using the toggle buttons at the top of the Projects page.

## Project statuses

Every project moves through a lifecycle:

- **Planning** -- Work has not started yet.
- **Active** -- Currently in progress.
- **On Hold** -- Temporarily paused.
- **Completed** -- All work is done.
- **Cancelled** -- Project was cancelled.

## Project dashboard

Click into any project to see its dashboard. This gives you a summary of:

- Overall completion percentage
- Upcoming tasks and deadlines
- Recent activity and updates
- Contract value and billing status
- Linked bids, pay apps, and change orders

Use the tabs within a project to navigate to phases, tasks, documents, pay apps, and field logs.
